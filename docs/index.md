# SocData

Unified, simple access to heterogeneous social science datasets.

- Direct adapters: Eurostat (API-backed)
- Semi-automatic: scripted recipes (e.g., WVS after manual download)
- Manual ingestion: parse local files from archives like ESS, ICPSR, ISSP

## Key features
- Consistent Python API and CLI
- Caching and reproducibility manifests
- Parsers for CSV/TSV, Stata (.dta), SPSS (.sav/.zsav)
- Parquet export
